## 前言

欢迎来到个人日程安排微信小程序的实现+SSM项目！本项目旨在帮助用户更好地管理个人日程安排，通过微信小程序的形式，方便快捷地记录和规划日常生活。以下是本项目的详细介绍。

## 内容介绍

本项目基于Java语言和SSM框架（Spring、SpringMVC、MyBatis），采用微信小程序作为前端展示，实现了一个功能齐全的个人日程安排应用。用户可以通过微信小程序创建、修改和查看日程安排，同时支持多端同步，让您的日程管理更加便捷。

## 技术介绍

- 语言：Java
- 使用框架：Spring、SpringMVC、MyBatis，微信小程序
- 前端技术：JS、Vue、CSS3，Uniapp
- 开发工具：IDEA/Eclipse，Uniapp
- 数据库：MySQL 5.7/8.0
- 数据库管理工具：phpstudy/Navicat
- JDK版本：jdk1.8
- Maven: apache-maven 3.8.1-bin
- 前端环境：Node.Js 12\14\16
